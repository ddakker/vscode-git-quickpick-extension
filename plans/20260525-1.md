## 2. 최초 커밋 파일 목록 안 보이는 버그 수정 (`src/git/queries.js`)

**원인:** `git diff-tree`는 parent 커밋이 없는 최초 커밋(root commit)에서 아무것도 반환하지 않음

**수정:** `--root` 플래그 추가
```
before: ['diff-tree', '--no-commit-id', '-r', '--no-renames', '--name-status', hash]
after:  ['diff-tree', '--root', '--no-commit-id', '-r', '--no-renames', '--name-status', hash]
```

**효과:** 일반 커밋에는 영향 없고, 최초 커밋에서도 추가된 파일 목록 정상 표시
